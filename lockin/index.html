Lock In: Focus + Screen Time

A personal productivity app for managing screen time

About the App

Lock In helps individuals focus by temporarily blocking distracting apps like social media during focus sessions. Users select which apps to restrict, set a duration, and stay productive without distractions. The app uses Apple’s Screen Time API for personal device management, ensuring full privacy and on-device processing.

Privacy

This app does not collect or share any usage data. All app usage metrics are processed and displayed locally on the user’s device.
